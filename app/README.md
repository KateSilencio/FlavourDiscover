# FlavourDiscover 
/************* Файл будет редактирован **************/

Android приложение для поиска рецептов с использованием Spoonacular API.

##  Начало работы

### Предварительные требования
- Android Studio 2022.3.1 или выше
- JDK 11
- Spoonacular API ключ

### Настройка API ключа

1. Скопируйте файл `develop.properties.example` в `develop.properties`
2. Замените `your_api_key_here` на ваш Spoonacular API ключ
3. Файл `develop.properties` автоматически игнорируется Git
